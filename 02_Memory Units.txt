컴퓨터에 데이터를 저장하는 공간은 휘발성 메모리와 비휘발성 메모리가 있다. 휘발성 메모리의 대표적인 예로 Random Access Memory인 RAM과 비휘발성 메모리의 대표적인 예로 Read Only Memory인 ROM이 있다. ROM에는 CD-Rom, DVD-ROM 혹은 메인보드 Bioas에 박혀 있는 rom, HDD 등이 있다.

앞으로 메모리라고 말하는 것은 휘발성 메모리인 RAM을 의미한다. RAM은 HDD나 CD와 달리 매우 속도가 빠르다. 왜냐하면 RAM의 경우 데이터에 랜덤하게 접근할 수 있는데 HDD나 CD는 순차 접근 해야하기 때문이다.

이런 매우 빠른 메모리의 특성 때문에 컴퓨터는 대부분의 데이터들을 메모리에 보관해 놓고 작업을 한다. 물론 중요한 데이터는 날아가지 않도록 틈틈히 HDD에 저장한다.

컴퓨터 한 개의 메모리 소자는 0 혹은 1의 값을 보관할 수 있다. 이 이진수 한 자리를 가리켜 bit라고 한다. 따라서 1개의 비트는 0또는 1의 값을 보관할 수 있다. 그러나 이는 너무 작은 양이다. 보통 우리는 1보다 큰 수를 다루기 때문이다. 따라서 8개의 bit를 묶어서 byte라고 부른다. 8bit는 1byte이다.

특별히 4bit를 nibble이라고 하지만 잘 쓰이지 않는다.

8bit로 나타낼 수 있는 수, 다시 말해 8자리 이진수로 나타낼 수 있는 최대의 수는 0000 0000 ~ 1111 1111  = 0 ~ 255 = 0 ~ 0xFF 로 총 256개(2^8개)를 나타내게 된다.

컴퓨터에서 연산을 담당하는 cpu에는 register라는 작은 메모리 공간이 이는데 이곳에 값을 불러다 놓고 연산을 수행하게 된다. a + b를 연산하기 위해서 a와 b의 값을 어디다 적어놓아야 a + b를 할 수 있다. cpu에서 연산을 위해 잠시 써놓은 부분이 레지스터이다. 이런 레지스터의 크기는 컴퓨터 상에서 연산이 실행되는 최소 단위라고 볼 수 있고 이 크기를 'word'라고 한다.
32 bit 컴픁 시절에는 1word가 32bit (=4byte)이지만 64bit 컴퓨터의 경우 1 word = 64 bit = 8byte 이다.
